CSS

Cascading Style Sheets - Folhas de Estilo em Cascata - tem por objetivo realizar a estilização de conteúdo de documentos HTML. Podem definir cores, fontes, espaçamento dentre outros.

Sintaxe:

seletor {

propriedades: valor; /*comentário*/

propriedades-subpropriedades: valor;

}


Exemplo:

p {

color: blue;

text-align: center;

}

Há três formas de utilizar o CSS

INLINE

  • Os códigos CSS são definidos dentro de uma tag HTML
  • Afetam somente a tag em que foi definida

INCORPORADO

  • Os códigos CSS são definidos no cabeçalho, dentro da tag head
  • Afetam somente o documento em que foi definido

EXTERNO

  • Os códigos CSS são definidos em um arquivo externo
  • Afetam todos os documentos que estão linkados ao CSS
