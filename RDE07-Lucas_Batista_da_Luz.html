Branches

São ramificações de um projeto de software ou documento e tem como objetivo o desenvolvimento paralelo de:

  • novas funcionalidades
  • correção de problemas
  • alteração de código

Branch Master ou Main

braço principal de um projeto em um repositório

Branch Secundária

braço secundário de um projeto em um repositório

Como criar

  1. Clicar em New Branch

  2. Colocar um Nome e clicar em Crete Branch

  3. Publicar a nova branch no GitHub

Hospedagem

O GitHub Pages oferece hospedagem gratuita de sites. Basta criar uma conta em um repositório público e a URL será:

https://nomeusuario.github.io/repositorio
