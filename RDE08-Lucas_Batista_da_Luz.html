Tag table

A tag table é usada quando se deseja criar uma tabela em duas dimensões, contendo linhas e colunas

São muito utilizadas para apresentar dados de forma organizada

Onde as tags < tr > representam uma linha da tabela

As tags < td > representam uma célula da tabela

Sintaxe:


Tag del

Representa uma parte de um texto que foi excluída de um documento

Exemplo:

Este texto foi deletado

Sintaxe:

< p >< del >Este texto foi deletado< /del >< /p >


Tag ins

Representa uma parte de um texto que foi inserida no documento

Exemplo:

Este texto foi inserido

Sintaxe:

< p >< ins >Este texto foi inserido< /ins >< /p >


Tag sub

Esta tag define um texto subscrito, um texto que aparece cerca de meio caractere abaixo da linha de escrita normal, geralmente renderizado pelo navegador a uma fonte menor

Exemplo:

Este texto contém texto subscrito na tela

Sintaxe:

< p >Este texto contém texto < sub >subscrito< /sub > na tela< /p >


Tag blockquote

Esta tag é usada quando se tem uma citação longa de outra fonte, onde os navegadores recuam esta tag do restante do documento de texto

Exemplo:

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Sintaxe:

< blockquote cite="https://pt.wikipedia.org/wiki/HTML" >

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

< /blockquote >


Tag q

Usada para citações curtas, onde o navegador insere aspas duplas em volta do texto citado

Exemplo:

A tradução de HTML é Linguagem de Marcação de Hipertexto

Sintaxe:

< p >A tradução de HTML é

< q >Linguagem de Marcação de Hipertexto< /q >

< /p >


Tag abbr

É usada para abreviaturas ou acrônimos

Exemplo:

A tradução de HTML pode ser vista abaixo

Sintaxe:

< p >A tradução de < abbr title="Linguagem de Marcação de Hipertexto" > HTML < /abbr > pode ser vista abaixo< /p >


Tag address

Define as informações de contato do autor ou proprietário de um documento ou artigo

Estas informações podem ser um endereço de e-mail, um site, endereço, número de telefone, entre outros

O texto dentro da tag é renderizado pelo navegador em itálico e os navegadores sempre darão uma quebra de linha antes e depois do texto

Exemplo:

Written by Jon Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

Sintaxe:

< address >

Written by Jon Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

< /address >


Tag cite

Define o título de um trabalho, como um filme, um livro, um poema, uma música

Geralmente o texto dentro da tag cite é renderizado em itálico

Exemplo:

O grito por Edward Munch. Pintado em 1893.

Sintaxe:

< p >< cite >O grito>< /cite >por Edward Munch. Pintado em 1893.< /p >


Tag bdo

Esta tag é usada para substituir a direção do texto em questão

Exemplo:

Este texto vai da direita para a esquerda

Sintaxe:

< p >< bdo dir="rtl" > Este texto vai da direita para a esquerda < /bdo >< /p >


Tag code

É usada para definir um trecho de código de computador, onde o texto é exibido no navegador em fonte monoespaçada padrão

Exemplo:

A propriedade CSS background-color define a cor de fundo de um elemento

Sintaxe:

< p >A propriedade CSS < code >background-color< /code >define a cor de fundo de um elemento< /p >


Tag pre

Esta tag define um texto pré-formatado, assim um texto dentro desta tag será exibido em uma fonte de largura fixa e preservando todos os espaços e as quebras de linha exatamento como escrito no código-fonte do html

Exemplo:

                O texto no elemento pre
                é mostrado com uma fonte
                de largura fixa, e preserva  
                os      espaços e
                quebras de linha

Sintaxe:

< pre >

                O texto no elemento pre
                é mostrado com uma fonte
                de largura fixa, e preserva  
                os      espaços e
                quebras de linha

< /pre >


Tag kbd

É usada para definir a entrada do teclado, onde seu conteúdo é mostrado em uma fonte monoespaçada padrão do navegador

Exemplo:

Pressione Ctrl + C para copiar um texto(Windows).

Sintaxe:

< p >Pressione < kbd >Ctrl< /kbd > + < kbd >C< /kbd > para copiar um texto(Windows).< /p >
